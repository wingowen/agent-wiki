## Who it is

Ken Aizawa 是 Anthropic 公司研究部门的 AI 研究员，主要从事 Agent AI 系统的研究与开发工作。

## Key facts

1. **主要贡献**：撰写了《Writing effective tools for agents — with agents》一文，该文是 Anthropic 关于如何为 AI Agent 设计高效工具的深度实践指南。

2. **研究领域**：
   - Agent 工具设计与评估
   - 上下文工程（Context Engineering）
   - 多 Agent 系统协作

3. **合作团队**：该文由 Anthropic 多个团队协作完成，包括研究部、MCP 团队、产品工程部、设计部和应用 AI 部。
