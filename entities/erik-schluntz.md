# Erik Schluntz

## What it is

Erik Schluntz 是 Anthropic 的工程师，也是《Building effective agents》这篇奠基性文章的作者之一。该文章总结了 Anthropic 在构建 LLM Agent 方面的核心经验和最佳实践。

## Key facts

- 与 Barry Zhang 共同撰写了《Building effective agents》
- 文章发表于 2024-12-19，成为 Agent 设计领域的重要参考文献
- 强调简单、可组合的模式而非复杂框架

## Notes

文章提出的核心观点包括：
- 最成功的 Agent 实现使用简单、可组合的模式
- 应优先使用直接 LLM API，只有在需要时才引入框架
- Workflows 适合定义明确的任务，Agents 适合需要灵活性的场景
