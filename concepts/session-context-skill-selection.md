# Session Context 影响技能选择

## 定义

Session Context（会话上下文）是指在用户与 Agent 对话过程中积累的历史信息，包括之前的消息、工具调用记录、执行结果等。这些上下文信息会被模型参考来影响后续的决策和响应生成。

## 关键问题

### 1. 会话历史管理问题

- **失败记录持久化**：工具调用失败的结果会被保存到会话历史（SessionManager 类）中
- **历史影响**：模型在生成回复时会参考会话历史，可能导致重复尝试相同失败的方法
- **缺乏智能过滤**：系统没有对失败的工具调用进行智能标记或过滤

### 2. 技能提示问题

- **技能信息不足**：技能提示中没有包含工具的历史成功/失败率
- **上下文缺失**：模型无法根据当前上下文判断某个工具是否适合当前任务

### 3. 上下文管理问题

- **历史压迫**：大量历史信息可能压迫当前决策
- **上下文稀释**：关键信息可能被历史记录稀释

## 关键解决方案

1. **会话历史压缩与摘要**：实现历史压缩和摘要功能，减少历史记录的压迫
2. **技能提示增强**：在技能提示中添加工具的成功率信息和适用场景
3. **上下文管理优化**：添加上下文重置机制和优先级管理
4. **错误处理优化**：实现智能错误处理和降级策略
