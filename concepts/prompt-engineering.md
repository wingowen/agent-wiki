## Definition

Prompt Engineering（提示工程）是设计和优化提示词以引导 LLM 生成期望输出的技术和实践。

## Key Techniques

### Few-shot Learning

通过示例演示期望的输入输出格式。

### Chain-of-Thought

引导模型逐步推理，提高复杂任务的准确率。

### System Prompt Design

设计系统级指令，控制模型行为、角色和能力边界。
